# Contributing to React Projects

Thank you for your interest in contributing! This repository contains multiple React projects.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** for your feature or fix: `git checkout -b features/my-new-feature`
4.  **Make your changes**, ensuring you are in the correct project directory (`caffiend`, `pokedex`, or `todo-app`).
5.  **Test your changes** by running `npm run dev` in the respective directory.
6.  **Commit your changes**: `git commit -m 'feat: add some feature'`
7.  **Push to the branch**: `git push origin feature/my-new-feature`
8.  **Open a Pull Request** against the main branch.

## Project Structure

*   `caffiend/`: A caffeine tracking app (React + Vite + Firebase).
*   `pokedex/`: A Pokemon explorer (React + Vite + PokeAPI).
*   `todo-app/`: A task manager (React + Vite + LocalStorage).

## Style Guide
*   Use `fanta.css` variables where possible.
*   Keep components small and functional.
